# katelovescode.com/katelovescode.github.io
## Kate Donaldson's CV Website

# Synopsis

CV website to provide a summary (with examples) of my skills, abilities, and accomplishments.

# Tools and Dependencies

A good engineer knows when to use a library or tool so she doesn't have to reinvent the wheel. To assist with standing up an app and successfully deploying it with automated build pipelines, I'm using a variety of tools, including Jekyll (with a theme that depends on Gulp), and Github Pages.  I develop using Visual Studio Code, a variety of helpful `git` and `bash` shortcuts I've collected over the years, and `git-flow-avh` to manage my git-flow feature branching workflow.

# Installation

To run locally for development:
- `git clone`
- `cd katelovescode.com`
- `npm install`
- `gulp`
